# Mini Ticketing System UI

A lightweight support ticket management dashboard built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Ticket Management: Create, update, delete, and track support tickets
- Ticket Details: View details, add comments, upload/download attachments
- Dashboard: Statistics overview, search, filter, and sort tickets
- Data Persistence: Ticket data stored in `localStorage`

## Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

## Prerequisites

- Node.js `>=20.9.0`
- `npm`
- `nvm` (recommended, because `.nvmrc` is included)

## Getting Started

From the repository root:

```bash
cd mini-ticketing-system-ui
nvm use
npm install
npm run dev
```

Then open:

- http://localhost:3000
